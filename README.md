# Easy Fitness - Professional Trainer-Trainee App

**Live Demo**: https://fitness-tracker-mocha-mu.vercel.app

## Quick Start

```bash
# Clone the repository
git clone [repository-url]
cd fitness-tracker

# Install dependencies
npm install

# Start development server
npm start

# Open http://localhost:3000 to view the app
```

## Available Scripts

### Development
- `npm start` - Run development server (localhost:3000)
- `npm test` - Run test suite
- `npm run build` - Build for production

### Deployment
The app is automatically deployed to Vercel on every push to the main branch.

## Tech Stack

- **Frontend**: React 19.2 + TypeScript
- **Styling**: CSS Custom Properties + Glassmorphism
- **Storage**: Local Storage (browser-based)
- **Deployment**: Vercel with auto-deployments
- **Design**: Mobile-first responsive with iOS container
